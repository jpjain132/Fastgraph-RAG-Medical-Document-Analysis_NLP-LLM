# 📌 Project Overview

This project implements an advanced GraphRAG-based document intelligence pipeline for:

1. Medical Clinical Document Analysis
2. Literary Knowledge Graph Analysis

The system combines:

- FastGraphRAG
- Local LLMs using Ollama
- Knowledge Graphs
- Neo4j Aura
- Biomedical NLP
- Semantic Retrieval
- Hallucination Detection
- NLP Evaluation Metrics

to transform unstructured documents into semantically searchable knowledge graphs.

The project demonstrates how Retrieval-Augmented Generation (RAG) can be enhanced using graph-based reasoning and entity relationships instead of relying only on vector similarity search.

# 🧠 Hallucination Detection Logic

Hallucinations are detected by verifying whether generated entities exist in the original source medical report.

Logic:
```python
if entity not in original_text.lower():
    hallucinated_relations += 1
```

Formula:
```python
hallucination_rate =
(
    hallucinated_relations
    /
    total_generated_relations
) * 100
```

This helps evaluate factual consistency of GraphRAG responses.
